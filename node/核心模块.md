## commonjs模块规范
在node中的js还有一个很重要的概念：模块系统
+ 模块作用域
+ 使用require方法用来加载模块
+ 使用exports接口对象用来导出模块中成员

### 加载require

```js
var 自定义变量名称 = require('modlue');
```

### 导出exports

```js
//导出多个成员
exports.a = 12;
exports.b = 'hello';
exports.d = {
    foo: 'bar'
}
module.exports = {
    a,
    b,
    d,
}
```

```js
//导出单个成员,拿到的是字符串，函数
module.exports = 'hello'
```
